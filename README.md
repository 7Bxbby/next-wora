# next-wora

Next-Wora (Next.js - write once, run anywhere) is a bundled starter template for building multi-platform Next.js apps (web + PWA + native).

## Quick start
Create a new project with the CLI:
```
npx create-next-wora@latest my-app
```

## Packages
- `packages/create-next-wora` — CLI that scaffolds the bundled templates.
- `packages/create-next-wora/templates/latest` — Next WORA starter (Next.js App Router + PWA + Capacitor shell).
